# Gemini Canvas Mission

Gemini Canvas 미션 결과물을 관리하는 저장소입니다.

## 목적

- Gemini Canvas로 제작한 웹앱 결과물을 기록하고 공유합니다.
- 앱별 요구사항, 배포 링크, 회고를 일관된 형식으로 남깁니다.

## 🚀 프로젝트 소개

Gemini Canvas를 활용하여 제작된 실용적인 웹 도구 모음입니다.

### 1. 정콩이의 뽀모도로 (`1. jeongkong_pomodoro.html`)

- **설명**: 집중력을 높여주는 화사한 파스텔 톤의 뽀모도로 타이머입니다.
- **주요 기능**: 드래그 방식의 직관적인 시간 설정, 백색소음/ASMR BGM 선택 기능, 집중 및 휴식 모드 전환.

### 2. 자바타자연습 (`2. java_type_training.html`)

- **설명**: 개발자를 위한 몰입형 타이핑 훈련 도구입니다.
- **주요 기능**: Java 주요 키워드 및 실제 코드 스니펫 , 실시간 WPM/정확도/오타 측정, 취약한 키 분석 기능.

### 3. 면접 준비 트레이닝 (`3. interview_training.html`)

- **설명**: 면접 대비를 위한 학습 도구입니다.
- **주요 기능**: 면접 단골질문인 Top 50을 선정하였고, 사용자가 직접 질문을 추가하거나 삭제할 수 있습니다.

### 4. 냉털 밀프렙 (`4. naengteol.html`)

- **설명**: 자취 초보를 위한 냉장고 털이 레시피 설계 도구입니다.
- **주요 기능**: 보유 재료/도구/양념 기반 레시피 생성, AI(Gemini) 기반의 맞춤형 밀프렙 가이드, 영양 성분 정보 제공.
